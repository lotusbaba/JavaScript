Preguntas

  1. Declaración recomendada para la creación de arrays en JavaScript.

    2. Se recomienda declarar arrays utilizando los corchetes [] en lugar de la llamada a new Array(), definiendo el array de forma literal:

    var array = ["Apple", "Orange", "Donkey", ...];
  2. pop() y push()

    1. Utilidad

      2. El método pop() sirve para eliminar el útlimo elemento de un array.
      El método push() sirve para añadir elementos al final de un array.

    2. ¿Modifican el array original?

      3. Si, modifican el array añadiendo o eliminando el elemento.

    3. ¿Devuelven algún valor?

      4. Si, el método pop() devuelve el elemento eliminado, y push() devuelve la nueva longitud del array.

  3. shift() y unshift()

    1. Utilidad

      2. El método shift() sirve para eliminar el primer elemento de un array.
      El método unshift() sirve para añadir elementos al principio de un array.

    2. ¿Modifican el array original?

      3. Si, modifican el array añadiendo o eliminando el elemento.

    3. ¿Devuelven algún valor?

      4. Si, el método shift() devuelve el elemento eliminado, y unshift() devuelve la nueva longitud del array.

  4. De los cuatro métodos anteriores, indica los recomendados para la inserción/eliminación de un elemento y por qué.

    5. Los métodos recomendados para la inserción/eliminación de un elemento son pop() y push() porque son más rápidos al no tener que reasignar las posiciones del resto de los elementos del array.

  5. join() y split(), ¿son métodos de String o de Array?

    6. join() es un método del objeto Array, y split() es un método del objeto String.

  6. splice()

    1. Utilidad

      2. El método splice() cambia el contenido de un array eliminando elementos existentes y/o añadiendo nuevos elementos.

    2. Indica significado del primer argumento admitido: índice, cantidad o contenido

      3. Índice donde se comenzará a cambiar el array.

    3. Indica significado del segundo argumento admitido: índice, cantidad o contenido

      4. Cantidad que indica el número de elementos del array que se tienen que eliminar. Si es igual a 0, no se eliminará ningún elemento.

    4. Indica significado del resto de argumentos admitidos: índice, cantidad o contenido

      5. Los elementos que se agregaran al array. Si no se especifica ningún elemento, splice() solamente eliminará elementos del array.

    5. ¿Modifica el array original?

      6. Si, modifican el array añadiendo o eliminando los elementos.

  7. slice()

    1. Utilidad

      2. El método slice() devuelve una copia de una parte del array dentro de un nuevo array.

    2. Indica significado del primer argumento admitido: índice, cantidad o contenido

      3. Índice donde empieza la extracción.

    3. Indica significado del segundo argumento admitido: índice, cantidad o contenido

      4. Índice que marca el fin a la extracción.

    4. Indica significado del resto de argumentos admitidos: índice, cantidad o contenido

      5. El método slice() no tiene más argumentos.

    5. ¿Modifica el array original?

      6. No, el array original no se modifica.

  8. Un Array es un Object:

    1. Indica resultado de typeof []

      2. Object

    2. Método forEach() de un array. Indica utilidad y si funciona en todos los navegadores. Entrega un pequeño código que demuestre su uso.

      3. El método forEach() ejecuta la función indicada una vez por cada elemento del array.

      Sintaxis:

      array.forEach(callback[, thisArg])

      Ejemplo de uso:

      function logArrayElements(element, index, array) {
	console.log("a[" + index + "] = " + element);
}
var array = [2, 5, 9];
array.forEach(logArrayElements);

      El método forEach() es compatible con todos los navegadores excepto Opera Mini e IE8/9.
